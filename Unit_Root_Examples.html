Classic Unit Root Test:

ADF Test:

    ADF test is made with using function from own GIT reposotory:

    if(!"devtools" %in% installed.packages()) install.packages("devtools")

UnitRootGIT='https://raw.githubusercontent.com/kamilbanas85/Unit-Root-Test---ADF/master/UnitRootFunctions.R'

devtools::source_url(UnitRootGIT)

    The ADFUnitRoot function based on urca::ur.df function (https://cran.r-project.org/web/packages/urca/urca.pdf).

    ADFUnitRoot( TS or XTS object,
             type = c("trend","drift","none", 
             selectlags = c("AIC","BIC","Fixed"),
             LagsNumber = 30)

    Example:

      plot(XTSobject)

      ADF test with trend and drift:

           ADFUnitRoot( XTSobject,
                type = "trend", 
                selectlags = "AIC",
                LagsNumber = 30)

KPSS Test:

    One response to the low power of ADF tests in the presence of near unit root stationarity was the test of Kwiatkowski, Phillips, Schmidt, and Shin (1992) (KPSS) http://blog.eviews.com/2019/04/seasonal-unit-root-tests.html

    KPSS test null hyphotesis: a time series is stationary around a mean or linear trend.

    tseries::kpss.test(XTSobject, null = "Level", lshort = TRUE)

    p-value < 0.01 dosn’t support null hypothesis, so null is rejected, Time Series has a unit root.

Seasonal Unit Roots Tests

    https://homepage.univie.ac.at/robert.kunst/season12.pdf
    “Deterministic versus stochastic seasonal fractional integration and structural breaks”
    https://www.jalobe.com/doc/uroot.pdf
    https://homepage.univie.ac.at/robert.kunst/season12.pdf

    Three main types of seasonality in time-series processes:

    • Deterministic seasonal processes: Seasonality is constant over time. It can be described by seasonal dummy varables.

    • Seasonal stochastic stationary processes: a change of a seasonal component(s) across time is/are stationary process - with ARMA model (the forecast in any given season is a function of past season values).

    • Seasonal stochastic processes with a seasonal unit root - series has a seasonal unit root, it means that the seasonal component(s) change across time is/are random walk (current season = last season + WN).

    The Hylleberg-Engle-Granger-Yoo (HEGY) statistics for testing the null hypothesis: seasonal unit roots exists

    plot(DataFrameEx_TS)

    uroot::hegy.test(DataFrameEx_TS,
              #  deterministic = c(0,0,0),
                 lag.method = "BIC", 
                 maxlag = 20)

    P-value < 0.01 of join F-statistic for seasonal frequency (365). The null hypothesis is rejected. Time Series is seasonal stationary.

Unit Roots for Time Series with Structural Breaks

    The most popular tests:

    • Zivot-Andrews (1992) unit root test: a test, which allows a break at an unknown point in either the intercept, the linear trend or in both.

    • Narayan and Popp (2010) unit root test: a test with two structural breaks in level and slope at unknown time.

    • Lee and Strazicich (2013, 2003) LM tests with one and two structural breaks.

    • Enders and Lee Fourier (2012) ADF and LM tests.
